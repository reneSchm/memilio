Detailed Description

template<typename CategoryTag>
class mio::Index< CategoryTag >

An Index with a single template parameter is a typesafe wrapper for size_t that is associated with a Tag.

It is used to index into a CustomIndexArray

CustomIndexArray<Tag1, Tag2> a; a[{Index<Tag1>(0), Index<Tag2>(14)}]

Will retrieve the element associated with the indices 0 and 14 for the Tags Tag1 and Tag2 respectively.

Optionally, the tag can be derived from Index to shorten the notation:

struct Tag1 : public Index<Tag1>{ Tag1(size_t val) : Index<Tag1>(val) {} }; struct Tag2 : public Index<Tag2>{ Tag2(size_t val) : Index<Tag2>(val) {} }; CustomIndexArray<Tag1, Tag2> a; a[{Tag1(0), Tag2(14)}]
