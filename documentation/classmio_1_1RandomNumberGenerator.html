Detailed Description

Models a uniform_random_bit_generator.

Keeps track of its seeds so they can be logged or set. The generated sequence can be segmented into blocks that can help to reproduce simulations involving random numbers by reliably generating a specific part of the sequence or to assign each thread/process a different sequence.

See also
thread_local_rng for a static instance.

◆ forward_to_block()

void mio::RandomNumberGenerator::forward_to_block ( size_t  block_idx)

Forward to block index i.

Skips numbers in the generated sequence up to the beginning of the block. The next number generated will be element block_size * i of the random sequence. This operation is O(n), where n is the number of skipped samples, so choose the block size with care.

Parameters
iblock index. May not be a block that is already passed or started.
